# TodoApp

A modern Todo application built using SwiftUI, SwiftData, and XCTest.

## Features

- Create, update, and delete todos
- Mark tasks as completed
- Search todos by title or description
- Local persistence using SwiftData
- MVVM architecture
- XCTest unit testing
- Real-time SwiftUI updates

## Technologies

- SwiftUI
- SwiftData
- Observation Framework
- XCTest
- MVVM Architecture

## Requirements

- iOS 17+
- Xcode 16+
- Swift 5.9+

## Run Project

1. Clone repository
2. Open in Xcode
3. Run the app using Simulator or physical device
